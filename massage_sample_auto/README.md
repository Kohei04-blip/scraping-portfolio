本リポジトリは、
クラウドワークスのスクレイピング案件応募用に作成した実績提示用ポートフォリオです。

5件のマッサージ店データを
・店名
・URL
・メールアドレス
・問い合わせフォームURL
・DR代替スコア（仮値）
として取得・整理しています。

🛠️使用技術
・Python3.x
・pandas
・CSV出力
・正規表現によるメール抽出（htmlは今回固定値）
・DR代替スコア：仮値（本番は被リンク数・類似指標で算出可能）

📃スクリプト説明：scraping_massage_sample.py
このスクリプトは、マッサージ店5件の情報をPythonの辞書データとして保持し、CSVに出力するものです。

実案件では、
・BeautifulSoup
・Google Maps API
・外部リンク数によるDR算出
　などに差し替えて運用できます。

▶️実行方法
python scraping_massage_sample.py
→同じフォルダにmassage_sample.csvが生成されます。
