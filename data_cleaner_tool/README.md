# データ整形ツール（CSVクリーニング）

Python を使って CSV データを整形するツールです。  
クラウドワークスなどの実務案件で最も多い「データ整理・整形」の要件を想定しています。

## 🔧 このツールでできること
- 重複行の削除(名前＋電話＋住所が同じものは1件にまとめ、メールアドレスがある行を優先して残します。)
- 電話番号の整形（数字以外の文字を除去）
- メールアドレスの抽出（正規表現）
- 空白行の削除
- 列名の統一（全て小文字、スペース除去）
- カテゴリ列の自動付与

## 📁 フォルダ構成
```
data_cleaner_tool/
├── README.md
├── data_cleaner.py
├── sample_input.csv
└── sample_output.csv（実行後に生成）
```

## ▶️ 実行方法

```
python data_cleaner.py
```

同じフォルダ内に `sample_output.csv` が生成されます。
